# Quita (Next.js completo)

## Rodar local
```bash
npm install
npm run dev
# http://localhost:3000
```

## Deploy Vercel
- Suba este diretório para um repositório GitHub.
- No Vercel: Import -> selecione o repo -> Deploy.
